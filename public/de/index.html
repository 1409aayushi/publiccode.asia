Public Money – Public Code

Warum wird Software, die von Steuerzahlern finanziert wird, nicht als Freie Software veröffentlicht? Von Allen bezahlter Code sollte Allen zur Verfügung stehen.

Steuerersparnis

Ähnliche Anwendungen müssen nicht jedes Mal von neu auf programmiert werden.

Zusammenarbeit

Bei großen Projekten können Expertise und Kosten zusammengelegt werden.

Für die Allgemeinheit

Anwendungen, die von Allen finanziert werde, müssen Allen zur Verfügung gestellt werden.

Innovation fördern

Dank transparenter Prozesse müssen Andere das Rad nicht neu erfinden.
